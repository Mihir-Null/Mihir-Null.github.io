# Mihir-Null.github.io

Deployment target for [**MiTerm**](https://github.com/Mihir-Null/MiTerm), my
personal website, served by GitHub Pages at [nullis.me](https://nullis.me)
from the [`docs/`](docs/) folder.

**This repository holds build output only.** The source — the terminal app, the
canonical `resume.org`, and the generators — lives in the MiTerm repo. `docs/`
is a static export published from MiTerm; do not edit it by hand. To change the
site, edit MiTerm and let its deploy workflow republish here.

Based on [LiveTerm](https://github.com/Cveinnt/LiveTerm).
